5️⃣ AdaBoost Model¶

🔍 Key Idea Reminder

  • Focuses on misclassified samples
  • Uses weights
  • Combines weak learners (usually stumps)
In [ ]:
ada_model = AdaBoostClassifier(
    estimator=DecisionTreeClassifier(max_depth=1),  # parameter name
    n_estimators=100,
    learning_rate=0.5,
    random_state=42
)

6️⃣ Gradient Boosting Model¶

🔍 Key Idea Reminder

  • Learns residuals (errors)
  • Uses gradient descent on loss function
  • More flexible than AdaBoost

🔟 Deep Understanding (Important)¶

AdaBoost Learning Behavior¶

  • Increases weight on:
    • hard samples
    • noisy samples ❗
  • Can over-focus on outliers

Gradient Boosting Learning Behavior¶

  • Learns:
    • residual patterns
    • smooth corrections
  • More stable than AdaBoost
