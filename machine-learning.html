Statistics for Machine Learning

  1. Statistics

    The science of collecting, analyzing, interpreting, presenting, and organizing data.

  2. Probability Distributions

    Mathematical functions describing the likelihood of different outcomes in random experiments.

  3. Descriptive Statistics

    Techniques for summarizing datasets using measures such as mean, median, variance, and standard deviation.

  4. Central Limit Theorem (CLT)

    A fundamental theorem stating that sample means tend to follow a normal distribution as sample size increases.

  5. Inferential Statistics

    Methods used to draw conclusions about populations based on sample data.

  6. Hypothesis Testing

    A formal framework to test assumptions about population parameters using statistical evidence.

  7. t-Test

    A statistical test used to compare the means of two groups.

  8. ANOVA Test

    Analysis of Variance used to compare means across three or more groups.

  9. Non-Parametric Tests

    Statistical tests that do not assume a specific data distribution.

  10. Statistical Modeling

    Building mathematical models to represent and analyze real-world data.

Machine Learning

  1. Fundamentals of ML

    Core principles of machine learning including learning paradigms, feature selection, model evaluation, and overfitting.

  2. Supervised Algorithms

    Algorithms trained on labeled data such as linear regression, decision trees, and support vector machines.

  3. Unsupervised Algorithms

    Techniques that discover hidden patterns in unlabeled data, such as clustering and dimensionality reduction.

  4. Reinforcement Learning

    Learning framework where an agent interacts with an environment to maximize cumulative reward.

  5. Support Vector Machine

    A powerful algorithm for classification and regression that finds the optimal hyperplane maximizing the margin between classes.

  6. Gradient Descent Method

    An optimization technique used to minimize the loss function by iteratively adjusting model parameters in the direction of the steepest descent.

  7. MLE & MAP

    MLE (Maximum Likelihood Estimation) estimates model parameters by maximizing the likelihood function, while MAP (Maximum A Posteriori) incorporates prior distributions.

  8. Polynomial Regression

    An extension of linear regression that models relationships as an nth-degree polynomial.

  9. Ridge, Lasso & Elastic Net

    Regularization techniques used to prevent overfitting and improve model performance.

  10. Principal Component Analysis (PCA)

    A dimensionality-reduction technique that transforms data into orthogonal components.

  11. Classification & Regression

    Methods for classifying data into categories and predicting continuous values.

  12. Naive Bayes

    A probabilistic classifier based on Bayes' theorem with an assumption of feature independence.

  13. K-Nearest Neighbors (KNN)

    A simple, non-parametric algorithm used for classification and regression based on proximity.

  14. Decision Tree

    A tree-based model that represents decisions and their possible consequences for classification and regression.

  15. Support Vector Machine

    Support Vector Machines find the optimal hyperplane that maximizes the margin between classes for classification and regression.

  16. Logistic Regression

    A classification algorithm used for binary outcomes, predicting probabilities based on the logistic function.

Deep learning

1. Optimization Methods

Techniques used to find the best possible solution or outcome by adjusting variables to minimize or maximize a particular function, commonly used in machine learning and operations research.

2. Deep Learning

A subset of machine learning involving neural networks with many layers (deep neural networks) that can learn and model complex patterns in large amounts of data.

3. Activation Function

A function in a neural network that determines whether a neuron should be activated or not, helping the network to learn complex patterns.

4. Convolutional Neural Networks (CNNs)

A class of deep learning neural networks, primarily used for analyzing visual data, that employ convolutional layers to automatically and adaptively learn spatial hierarchies of features from input images.

9. Ensemble learning

Ensemble learning in machine learning refers to techniques that combine the predictions from multiple models (learners) to improve the overall performance.
